A curated and automatically updated list of open-source repositories along with active issues across multiple projects. This repository helps developers quickly find issues to contribute to and explore trending problems in the open-source ecosystem.

📌 Overview

This project compiles issues from various GitHub repositories across different domains like:

Web Development
DevOps
UI/UX
Backend Systems
Documentation
Open Source Tools

The list is updated daily at midnight, ensuring fresh and relevant issues for contributors.

🎯 Purpose
Help beginners find "good first issues"
Provide a centralized place for open-source contributions
Save time searching across multiple repositories
Encourage participation in real-world projects
📂 Repository Structure

The list is organized by repository names, for example:

## repository-name

- Issue 1
- Issue 2
- Issue 3

Direct GitHub issue link
